# Auth Portal (Login & Registration)

This is an independent Next.js app for login and registration that mirrors the stack and theme from the main website.

## Getting Started

```
npm install
npm run dev
```

## Environment

Create `.env.local` with:

```
NEXT_PUBLIC_API_URL=https://chwh.store/backend/api
```

## Routes
- /            Home hub
- /login       Login form
- /register    Registration form
